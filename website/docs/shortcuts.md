# Keyboard Shortcuts

TickTock is designed to be fully controllable via keyboard. Master these shortcuts to track your time without touching your mouse.

## Global Shortcuts

These work even when TickTock is not the active window.

| Shortcut | Action |
| :--- | :--- |
| `⌥ + Space` | Toggle Show/Hide TickTock |

## App Shortcuts

These work when TickTock is focused.

| Shortcut | Action |
| :--- | :--- |
| `⌘ + N` | Create New Task |
| `⌘ + F` | Search Tasks |
| `⌘ + S` | Toggle Tracking (resumes last active task) |
| `⌘ + T` | Jump to Today |
| `⌘ + D` | Jump to Date |
| `⌘ + 1-9` | Start tracking task from Activity Log |
| `⌘ + A-Z` | Open Library task for editing |
| `?` | Show Help Dialog |
| `Esc` | Clear search / Unfocus |

## Modal Shortcuts

These work when a Task Modal is open.

| Shortcut | Action |
| :--- | :--- |
| `⌘ + S` | Start Tracking |
| `⌘ + ⌫` | Delete Task |
| `⌘ + Enter` | Save Changes |
| `Esc` | Close Modal |
